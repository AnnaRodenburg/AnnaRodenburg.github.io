Introduction

Bootstrap is an open-sourced CSS framework for the responsive web. It is to help an average coder with coding the perfectly designed websites.

Using bootstrap is effective and will make your website more attractive to the public with the many features available.To start using bootstrap you'll need a coding program such as Visual Studio Code and knowledge on how to use HTML and CSS.

Bootstrap Spacing

Spacing from bootstrap is used for responsive margin and padding classes and it changes the how the element looks. To use spacing in bootstrap, you have to assign responsive friendly values to any specific element of your choosing. Also you must pick a number between .25rem to 3rem. You must also use the !important tag at the ending.

Bootstrap Jumbotron

The jumbotron is a component on a website that can show information in a more proffesional manner and can span all of the viewpoints desired. To use the jumbotron, you must use the jumbotron . fluid class. This also makes the jumbotron the full width of the page and with no rounded corners.

Bootstrap Cards

The cards bootstrap feature can help layout the website in a very attractive way and let any viewer have an easier time finding important information.
